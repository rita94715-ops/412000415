# 🎰 我的第一個 Python 抽卡小遊戲

這是我在 GitHub 上的第一個開發專案！透過 AI 領航員的引導，我學習了如何從零開始建立一個抽卡模擬系統，並完成數位存檔。

## 🎮 遊戲簡介
這是一個基於 Python 撰寫的**隨機抽卡模擬器**。
它可以模擬單抽與十連抽的過程，並且會根據稀有度顯示不同的顏色反饋。

### 🌟 抽卡機率說明
| 稀有度 | 機率 | 顏色 |
| :--- | :--- | :--- |
| **UR** | 1% | 紅色 |
| **SSR** | 4% | 黃色 |
| **SR** | 15% | 紫色 |
| **R/N** | 80% | 預設 |
